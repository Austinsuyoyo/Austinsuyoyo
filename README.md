```c
struct engineer austin = {
    .role     = "Firmware Engineer",
```

- 🔧 I build **firmware and embedded systems** — BMC/BIOS platforms, Yocto-based Linux distros, and low-level board bring-up.
- 🖥️ Day-to-day: **OpenBMC** on AST2700, **Yocto / BitBake** recipes, **coreboot**, PECI / MCTP / SPDM, and a lot of time in the kernel & device tree.
- 🌱 Off the clock I ship **full-stack apps** (Nuxt / TypeScript) and glue everything together with **n8n** and **Cloudflare** automation.
- 📫 Find me at **[austinsuyoyo.com](https://www.austinsuyoyo.com)**.
